# React mini blog

A small blog application built with React.

This project was created to practice working with forms, validation and POST requests

## What the project demonstrates

- Creating and controlling forms in React
- Checking user input before sending
- Handling form submission
- Sending POST requests to an API
- Managing loading, success and error states
- Rendering dynamic lists of posts
- Conditional rendering in React (showing elements only when needed)
- Splitting logic into separate components

## Technologies used

- React
- useState
- useEffect
- JavaScript
- CSS
- REST API

## How to run the project

npm install
npm run dev
